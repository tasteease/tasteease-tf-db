# Tasteease - Manage AWS RDS With Terraform

Projeto de IaC para implementação de banco de dados PostgreSQL para AWA RDS.

```bash
  terraform init
  terraform plan
  terraform apply
```

## Techs

- Terraform
- AWS RDS
